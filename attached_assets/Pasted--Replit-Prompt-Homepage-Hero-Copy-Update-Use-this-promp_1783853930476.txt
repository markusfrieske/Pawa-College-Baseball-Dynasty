```text

```

```text

```

Use the existing component structure and styling. This is a copy/content update, not a full redesign.

```text

```

with:

```text

```


Replace the current paragraph with this copy:

```text
Build a solo or multiplayer college baseball dynasty with 3,700+ players across 149 schools. Recruit every week, manage your roster, play or report your series, track stats and standings, and chase the College World Series across 20 seasons.
```

## Primary Buttons

Use these two primary hero buttons:

```text
Get Started
View Rosters
```

Expected behavior:

- `Get Started` should route to the existing sign-up/login/get-started flow.
- `View Rosters` should route to the roster viewer or public roster page.

Use the existing route names in the app. Do not invent new pages if the routes already exist.

## Secondary Links

Below the primary buttons, show compact secondary links:

```text
View Leagues
Create Recruiting Class
Donate
```

Expected behavior:

- `View Leagues` routes to the public leagues list or league selection page.
- `Create Recruiting Class` routes to the recruiting class generator/wizard if it exists.
- `Donate` routes to the existing donation/support link if present.

If a route does not exist yet, leave the link hidden or disabled with a TODO comment. Do not create a broken link.

## Layout / Readability Requirements

The current hero image is visually strong, so the copy must remain readable on top of it.

Requirements:

- Keep text left-aligned.
- Do not place the text inside a large card.
- Use subtle text shadow if needed.
- Avoid a heavy black overlay that hides the background art.
- Make sure the body copy does not collide with the character art.
- Desktop headline can remain large and stacked.
- Mobile headline should scale down enough that it does not overflow.
- Do not use negative letter spacing.

Suggested mobile handling:

```css
font-size: clamp(2.5rem, 12vw, 5rem);
line-height: 0.95;
```

Only use this if it fits the existing styling system.
