# Replit Prompt: Update UCLA Default Roster Ratings From Screenshots - Batch 1

## Goal

Update the base/default UCLA roster so the first batch of UCLA players matches the provided PowerPros-style screenshots.

Only update:

- Numeric player attributes.
- Common ability grades/fields.
- Special abilities.

Do **not** change:

- Player names.
- Positions.
- Eligibility/class.
- Jersey numbers.
- Hometown/home state.
- Bat/throw handedness.
- Potential.
- Pitch mix.
- Pitcher-specific ratings for non-pitchers.
- Team metadata.
- Any non-UCLA players.

The UCLA default roster is in:


```ts

  ...
```

Use this mapping from screenshot labels to roster fields:

```text
弾道             -> trajectory

捕球             -> errorResistance

ケガしにくさ     -> grit
盗塁             -> stealing

回復             -> recovery
```

Use exact numeric values for main attributes.

For common ability grades that are shown as letters only, use the repo's existing common-grade conversion if one exists. If no canonical conversion exists, use these numeric anchors so the displayed grades match:

```text
S = 90
A = 80
B = 70
C = 60
D = 50
E = 40
F = 30
```

For players without `catcherAbility`, keep `catcherAbility: null` unless the screenshot has a catcher grade.
