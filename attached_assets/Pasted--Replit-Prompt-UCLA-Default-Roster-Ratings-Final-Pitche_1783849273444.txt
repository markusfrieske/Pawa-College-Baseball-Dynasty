## Velocity Conversion

The repo stores pitcher `velocity` as a 1-99 internal rating and displays it as KMH through `velocityToKMH()`:

```ts
velocityToMPH(rating) = round(82 + ((rating - 1) * 20) / 98)
velocityToKMH(rating) = round(velocityToMPH(rating) * 1.60934)
```

Use the internal `velocity` value that displays closest to the screenshot KMH.

## Pitch Mix Field Order

`pitchMix(1, [...])` uses this secondary pitch order:

```ts
pitchMix(1, [pitch2S, pitchSL, pitchCB, pitchCH, pitchCT, pitchSNK, pitchSPL, pitchFK, pitchSFF, pitchSHU])
```

Returned fields:

```text
pitchFB, pitch2S, pitchSL, pitchCB, pitchCH, pitchCT, pitchSNK, pitchSPL, pitchVSL, pitchFK, pitchSFF, pitchSHU
```

Important:

- `pitchFB` is binary. Fastball quality comes from `velocity`, `stuff`, and common abilities.
- `pitch2S`, `pitchFK`, and `pitchSFF` are binary in the helper.
- Most other pitch types use levels `0-7`.
- Match the screenshot pitch diagrams as closely as possible.
